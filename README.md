# Fast React component generator for Vscode

One-click generator to create new React components based on best practices.

It will create:

- A new folder
- - __tests__/{name}.tsx
- - {name}.tsx
- - use-{name}.ts
- - constants.tsx
- - type.tsx
- - styles.scss
- - index.ts

`name` must be a kebab-case string
